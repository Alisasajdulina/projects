это приложение для сбора и анализа данных о погоде из различных источников через API. Проект создан в рамках учебного курса по Data Science для освоения практических навыков работы с API, обработки данных и их сохранения в структурированном виде.

Цели
- Освоить работу с REST API
- Научиться обрабатывать JSON данные
- Сохранять данные в структурированном виде (CSV, JSON)
- Работать с виртуальным окружением Python

- Python 3.12
- API ключ Gismeteo (бесплатный на сайте)
