Описание проекта
Проект демонстрирует сбор данных о погоде через Gismeteo API для практического занятия по Data Science.

Цели
- Освоить работу с REST API
- Научиться обрабатывать JSON данные
- Сохранять данные в структурированном виде (CSV, JSON)
- Работать с виртуальным окружением Python

- Python 3.12
- API ключ Gismeteo (бесплатный на сайте)
